# MATLAB language

## Comments

### Single-line comments

A signle-line comment begins with **%**. Everything between **%** to the end of the lineis ignored when the script is executed.

```MATLAB
% This is a line comment
```

### Multi-line comments

A multi-line comments begins with **%{** and end with **%}**. Everything beteween **%{** and **%}**. is ignored when the script is execute.

```MATLAB
%{
    this is a mutl-line comments example:
    1. first line
    2. second line
}%
```

### Block comment

The first comment at the beginning of the script file is called a **block comment**. The **block comment** will be displayed in the **help** command.

It is also for the **lookfor** command searches through.
