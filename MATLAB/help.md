# MATLAB help system

## help command

* Enter **help** command will display all avalilable commands.

```MATLAB
help
```

* Enter **help command** will display the help information about this specific **_command_**

```MATLAB
help sum
```

## which command

```MATLAB
which sim
```

## lookfor command

```MATLAB
lookfor log
```

## doc command


```MATLAB
doc sum
```
